1. Объектно-ориентированное программирование (ООП) - это подход к написанию кода, основанный на концепции объектов, которые являются экземплярами классов и имеют свойства и методы.

2. Основными принципами ООП являются: наследование, инкапсуляция, полиморфизм и абстракция.

3. Инкапсуляция - это oсновной принцип ООП, который позволяет скрыть внутреннюю реализацию объекта от других объектов. Это достигается путем объединения данных и методов, которые работают с этими данными, в классе и предоставления доступа к ним только через определенные методы.

4. Класс в Java - это шаблон для создания объектов, который определяет свойства и методы, которые будут доступны для объектов этого класса.

5. Класс - это шаблон, а объект - это экземпляр класса, созданный в памяти компьютера. Каждый объект имеет свои уникальные свойства и может вызывать методы, определенные в его классе.
